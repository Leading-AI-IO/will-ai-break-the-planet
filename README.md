## 📖 概要

2026年、AIは社会の中心議題になった。とりわけ激しく語られているのは「雇用」だ——AIに仕事を奪われる、新卒が職に就けない、ホワイトカラーの中間層が崩れる。これらは近く、可視で、自分ごとの脅威だからこそ、世論もメディアも全力で反応する。

その一方で、ほとんど誰も賭け金を数えていない問いがある。**世界中のAI企業と資本と政府が数十兆円規模のAIインフラ投資に注力する過程で、いったいどれだけのCO2が排出され、それは地球に、そして人類に、最終的に何をもたらすのか。**

本書は終末論ではない。むしろ、この問題で最も権威ある公式見解——IEAの「データセンターのCO2は世界排出の約1%にすぎず、2030年頃にピークアウトする」という楽観的予測——を正面から受け止めることから始める。だが本書が問うのは「量」ではない。**「方向」と「時間」だ。**

そのピークアウトは、SMR・新規原子力・送電網・再エネが「予定通りに間に合う」という前提の上に立つ。予測ではなく、賭けである。そして、その賭けが「間に合う」とされる時間軸は、地球が不可逆な臨界点（ティッピングポイント）を越えるリスクが急騰する時間軸と、重なっている。

判定ではない。本書は、AIインフラ投資・気候の不可逆性・安全保障の帰結・人間の認知バイアスという、これまで別々に語られてきた**四つの議論を一本の線でつなぎ、「時間軸の衝突」という一つの構造として記述する**ことを目的とするOSS書籍である。
